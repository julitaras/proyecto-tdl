## Run app

First, must have install [`flutter`](https://flutter.dev/docs/get-started/install).

Second, install emulator and run it.

Finally,
in the root project: 

  **Get the packages:**
  ```sh
  flutter pub get
  ```

  **Run app:**
  ```sh
  flutter run
  ```
